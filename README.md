# Reqres API Automation Project

## Overview
This repository contains API Automation scripts for reqres get users project.

The automation covers the functionality of https://reqres.in/api/users?page=2 API.

## Prerequisites
Please ensure the following are installed:
- Katalon Studio (version 10.x.x or latest version is recommended)
- Java (compatible with katalon)
- Git

## How to setup
1. Clone repository
git clone https://github.com/fiqihz/Reqres-Users

2. Open project
- Open Katalon Studio
- Click Open Project
- Select the cloned project folder 

## How to run test
1. Navigate to Test Suites
2. Select List Users
3. Click Run using Web Service

## Test reports
Reports are automatically generated under /Reports folder
